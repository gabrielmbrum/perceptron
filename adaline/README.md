# ADALINE (Adaptive Linear Element)

Este repositório contém a implementação de uma rede neural **ADALINE**, utilizando a **Regra Delta** para o ajuste de pesos e minimização do Erro Quadrático Médio (EQM). O script permite a execução tanto em modo **Estocástico** quanto em **Lote (Batch)**.

## Como Executar

O script `adaline.py` processa arquivos de dados e gera resultados de treinamento e teste.

```bash
python3 adaline.py
```
